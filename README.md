# 🔐 Secrets Detector (VS Code Extension)

A powerful VS Code extension that automatically detects hardcoded secrets, API keys, and sensitive credentials in your codebase — helping you prevent accidental exposure before committing to version control.

---

## 🚀 Overview

Secrets Detector provides real-time security scanning inside your editor to identify potentially dangerous hardcoded values such as API keys, tokens, passwords, and database credentials.

It helps developers maintain secure coding practices and avoid accidentally leaking sensitive information into public repositories.

---

## ✨ Features

- 🔴 Real-time detection of secrets while coding  
- ⚠️ Instant warning notifications  
- 🔍 Manual scan command for full workspace analysis  
- 📋 Integration with VS Code Problems panel  
- ⚡ Lightweight and fast performance  

---

## 🔎 What It Detects

Secrets Detector can identify a wide range of sensitive data, including:

- AWS, Google Cloud, Firebase, and Azure keys  
- GitHub, GitLab, and NPM tokens  
- OpenAI, Stripe, PayPal API keys  
- Database URLs (MongoDB, PostgreSQL, MySQL, Redis)  
- Private keys and certificates  
- Hardcoded passwords and credentials  
- 50+ additional secret patterns  

---

## ⚙️ Usage

Secrets are automatically detected when you open or edit files.
